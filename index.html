How we work

Working with PCGsoft

A grounded process: understand first, then build the smallest system that solves the real problem.

  1. Understand the workflow

    Map how work actually happens — people, hand-offs, tools and constraints — before writing software.

  2. Identify the real problem

    Separate symptoms from causes so the solution targets friction that matters, not noise.

  3. Prototype the right solution

    Shape a clear direction early — enough to test assumptions without over-building.

  4. Build, test and improve

    Deliver maintainable software, validate it in use, and refine based on real operation.

The appropriate answer may be a website, web application, mobile application, automation — or a combination. PCGsoft investigates carefully and builds what fits; not every enquiry becomes a project.

About

An independent UK software studio

PCGsoft is an independent UK software studio founded by Phil Geran. It grew from first-hand experience of operational work and the belief that useful software should reflect how people actually work, not force people into a generic process.

The studio’s current work spans transport compliance, business systems, trustworthy AI infrastructure and selected creative applications.
